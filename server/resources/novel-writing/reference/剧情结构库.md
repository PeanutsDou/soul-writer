# 剧情结构库

> 每次分析小说后，提取其剧情结构和章节骨架，
> 归类到对应的模式中。
> 写作时可从库中选取结构模板进行编排。

---

## 一、开篇模式

### 1.1 穿越＋世界观突变型

**结构**：主角穿越/醒来 → 发现世界变了 → 接收记忆 → 识别核心矛盾 → 确定初步目标

**适用题材**：异世界、历史、玄幻、轻小说

**案例**：
- 《这次不当御兽师了》：醒来发现水产养殖→水系宠兽研究学，专业突变制造悬念
- 《大明草包探花》：穿越赶上会试放榜，发现自己完全不会八股文
- 《穿越魔教教主，但甲子荡魔》："历史区都去了秦汉唐宋，怎么我到了大元？"

**节奏模板**（前5章）：
```
第1章  穿越落地 + 世界观突变 → 悬念建立
第2章  确认处境 + 金手指觉醒/发现 → 希望感
第3章  验证金手指 + 初步尝试 → 信心建立
第4章  获取第一批资源/伙伴 → 动力生成
第5章  确立长线目标 → 期待感锚定
```

**优势**：世界观和主线在开局就清晰，读者理解成本低
**风险**：穿越理由和世界观如果不够新鲜，容易劝退
**关键成功因素**：穿越后的第一反应必须有特色（吐槽、冷静分析、恐慌——但不能平庸）

---

### 1.2 已知世界先知型

**结构**：主角意识到自己在已知作品世界 → 利用剧情先知优势 → 打破原本命运 → 抱大腿/提前布局

**适用题材**：诸天无限、同人、轻小说

**案例**：
- 《恐怖世界的道士》：知道双瞳电影全部剧情，提前布局阻止反派
- 《诸天：小李飞刀开始》：知道自己穿成原著活不过第一章的龙套，果断改变命运
- 《同时穿越：从牢九门开始操作》：知道咒术回战世界走向，同时警惕其他穿越者

**节奏模板**（前5章）：
```
第1章  确认世界 + 评估自身处境 → 紧迫感建立
第2章  利用先知做出关键决策 → 认知差爽
第3章  避开原著的致命陷阱 → 读者松一口气
第4章  建立新的人际关系/势力 → 改变原著线
第5章  埋下后续危机伏笔 → 长线期待感
```

**优势**：读者自带世界观认知，节省铺垫篇幅
**风险**：同人依赖读者对原著的了解，纯路人读者可能跟不上
**关键成功因素**：选择热门但有新鲜感的世界（双瞳这种冷门高分片 > 斗罗斗破这种被写烂的）

---

### 1.3 日常切入反转型

**结构**：最普通的日常场景 → 出现意外 → 发现秘密 → 被卷入事件

**适用题材**：都市、轻喜剧、轻小说

**案例**：
- 《姐姐，我也要当女仆吗？》：收租（日常）→遇到哑巴同学开口说话（意外）→发现双胞胎妹妹（秘密）→被误认为男朋友（卷入）

**节奏模板**（前5章）：
```
第1章  日常任务 → 偶遇异常 → 发现小秘密 → 钩子
第2章  秘密扩大 → 被迫卷入 → 互动冲突 → 喜剧效果
第3章  更多人物出场 → 关系复杂化 → 期待感建立
第4章  角色背景揭示 → 情感升温 → 读者站队
第5章  第一个小高潮/反转 → 确认追读
```

**优势**：零门槛入学，不挑读者
**风险**：开局缺乏紧迫感，在"黄金三章"的竞争中容易被淹没
**关键成功因素**：日常场景必须有新意（收租遇到同学是日常，但哑巴同学突然说话就是反转）

## 二、主线推进模式

### 2.1 科研/实验型推进

**结构**：设定目标 → 提出假设 → 设计实验 → 积累资源 → 执行实验 → 分析结果 → 迭代

**适用题材**：学霸文、科研文、御兽文、炼丹文

**案例**：
- 《这次不当御兽师了》：假设金纹锦鲤能进化成龙 → 四项条件分析 → 逐项攻克 → 鲤鱼跃龙门实验 → 成功

**章节节奏**：
```
铺垫章（信息）→ 资源章（积累）→ 实验章（过程）→ 高潮章（结果）
      ↓          ↓           ↓           ↓
  抛出假设   获得材料     执行方案    成功/失败/迭代
```

**优势**：逻辑清晰的推进感，读者参与感强（跟着主角一起解谜）
**风险**：容易写成说明书，需要有情绪高点穿插
**平衡技巧**：每2-3章信息章后加1章情绪/互动章作为缓冲

---

### 2.2 危机倒计时型推进

**结构**：暴露出未来危机 → 危机逐步逼近 → 主角各方周旋 → 在deadline前解决/逃脱

**适用题材**：历史文、诸天、阴谋类

**案例**：
- 《穿越魔教教主，但甲子荡魔》：张三丰要来甲子荡魔（全书倒计时）→ 手下想杀他（内部倒计时）→ 当前危机（老道士打上门）
- 《大明草包探花》：南北榜案即将爆发 → 朱元璋即将清洗 → 主角在风暴边缘周旋

**章节节奏**：
```
危机提示章 → 日常平静章（反差）→ 危机逼近章 → 应对章 → 危机升级章 → ...
    ↘              ↘              ↘           ↘          ↘
  读者紧张      暂时放松        紧张升级    希望出现     更紧张
```

**优势**：天然的追读驱动力——读者一直在等"那个时刻"到来
**风险**：如果倒计时太长（超过50章还没有兑现），读者会疲劳
**平衡技巧**：大倒计时里套小倒计时（张三丰是全书的，老道士是当卷的，手下暗算是当章的）

---

### 2.3 抱大腿/借势型推进

**结构**：主角实力不足 → 识别强者 → 用利益交换抱上大腿 → 借大腿之势积累自身

**适用题材**：诸天、武侠、轻小说

**案例**：
- 《诸天：小李飞刀开始》：用金丝甲换阿飞好感 + 剑法教学 → 借阿飞的武力在江湖立足

**章节节奏**：
```
发现大腿 → 制造接触机会 → 利益交换 → 获得庇护/教学 → 自身成长 → 关系深化 → 逐步独立
```

**优势**：解决"开局太弱怎么活"的难题，逻辑合理
**风险**：主角长期依赖大腿会显得无能
**平衡技巧**：大腿必须是"拐杖"而非"代打"——主角要借用大腿的资源/威慑力，但不能靠大腿解决所有问题

---

### 2.4 多方压力/平衡型推进

**结构**：主角身处多方势力的夹缝中 → 各方都有不同诉求 → 主角在夹缝中周旋 → 利用情报差平衡各势力

**适用题材**：权谋、历史、魔教/反派主角

**案例**：
- 《穿越魔教教主，但甲子荡魔》：手下想杀他当替死鬼 vs 外面正道要除魔 vs 元庭官方也在盯着

**章节节奏**：
```
A方施压 → 主角装怂 → B方施压 → 主角承诺 → A方察觉 → 危机升级 → 主角找到破绽 → 反击
```

**优势**：天然的戏剧张力，每章都有冲突
**风险**：对作者的谋略水平要求高，写不好主角像傻子
**平衡技巧**：至少给主角留一个"信息差优势"——他知道别人不知道的事，不然就没法破局

---

### 2.5 日常/互动型推进

**结构**：由角色关系和日常互动驱动的推进，弱化主线冲突，强化情感连接

**适用题材**：都市轻喜剧、日常向、轻小说

**案例**：
- 《姐姐，我也要当女仆吗？》：收租→遇到同学→发现秘密→被迫收留→日常互动→关系变化

**章节节奏**：
```
日常互动 → 小冲突 → 解决 → 感情升温 → 新人物登场 → 关系复杂化 → ...
```

**优势**：阅读压力小，读者粘性来自舍不得角色
**风险**：容易平淡，需要有持续的"新鲜事件"注入
**平衡技巧**：每5-8章需要一个"事件章"打破日常节奏（新人物登场、某个秘密被揭露等）

---

### 2.6 专业/职业线推进 ✨（2026Q1新增）

**结构**：专业事件触发 → 诊断/分析/方案 → 执行解决 → 获得认可/成就感 → 更复杂的案子

**适用题材**：医疗文、法律文、商战文、科研文、技术类

**案例**：
- 《重生08，我被确诊为医学泰斗》：改良Whipple手术震惊学界 → 研发广谱抗癌药 → 基因筛查突破 → 每个新案例都是一个小闭环
- 《首富从AI浪潮开始》：看到数据面板 → 利用信息差解决技术难题 → 商业扩张 → 更多复杂挑战

**章节节奏**：
```
铺垫章（问题出现）→ 分析章（诊断/方案）→ 执行章（手术/谈判/研发）→ 结果章（成功/迭代）
      ↓                    ↓                    ↓                    ↓
  患者/问题出现        主角制定策略        执行过程中的波折      情绪释放/新线索
```

**优势**：天然的解谜爽结构 + 真实的专业门槛 = 读者信任感强
**风险**：对作者的专业知识储备要求高，写错会被懂行的读者挑刺
**平衡技巧**：专业细节不用100%真实，但逻辑链条必须自洽——读者要的是"可信的专业感"而非"真实的教科书"

---

### 2.7 系统错位型推进 ✨（2026Q1新增）

**结构**：系统给出"不合理"的判断 → 主角质疑 → 验证 → 发现系统是对的 → 利用系统优势推进

**适用题材**：轻小说、都市、御兽、系统文

**案例**：
- 《这次不当御兽师了》：系统说"这是鲤鱼王"→主角不信→验证→真的进化了→用系统逻辑继续探索
- 《人在美利坚：我的叔叔堂吉诃德》：系统把叔叔当骑士→主角困惑→发现系统任务真的有奖励→开始利用系统

**章节节奏**：
```
错位建立章（系统说A是B） → 怀疑章（主角不信） → 验证章（发现真的可以）→ 利用章（获取好处）→ 深化章（更多错位）
       ↓                       ↓                      ↓                      ↓                   ↓
   读者好奇               读者代入质疑           反转惊喜             爽感释放           期待扩展
```

**与常规系统文的核心差异**：
- 常规系统文：系统是"全能的正确答案"
- 错位系统文：系统是"可能有问题的，但能用"——主角需要甄别系统给出的信息

**优势**：持续的"惊喜反转"——每次系统给出新"错误"判断，读者都会期待"这次又是什么"
**风险**：如果系统错位没有逻辑（纯粹为错而错），读者会觉得是作者在瞎编

---

### 2.8 海外生存/探险型推进 ✨（2026Q1新增）

**结构**：主角来到海外陌生环境 → 识别资源和危险 → 凭借独特优势立足 → 逐步扩张影响力

**适用题材**：都市海外文、异域文

**案例**：
- 《人在美利坚：我的叔叔堂吉诃德》：赴纽约→发现叔叔是疯子→系统任务→"第一天遵纪守法，第五天厌倦这个世界"
- 《医学美利坚：我靠恶魔度过斩杀线》：在美利坚行医→利用医疗系统漏洞→建立名声

**优势**：海外环境天然的"文化冲突"带来持续的新鲜事件
**风险**：需要作者对海外环境有一定了解，否则容易出戏

## 三、三幕式章节分布

分析7本书后，前10章的普遍分布规律：

### 3.1 标准三幕分配

```
第一幕（1-3章）：开篇 + 金手指 + 完成世界认知
  → 核心任务：让读者在3章内决定是否追读

第二幕（4-7章）：积累资源 + 训练/学习 + 推动主线
  → 核心任务：建立期待感，准备高潮

第三幕（8-10章）：高潮事件 + 完成首个循环 + 埋下更大伏笔
  → 核心任务：兑现第一波期待，给读者"看完一卷"的满足感
```

### 3.2 章节功能分类

每章可按功能标记，确保类型分布合理：

| 类型 | 比例 | 功能 | 案例 |
|------|------|------|------|
| **信息章** | 20-30% | 铺设设定、介绍人物、交代背景 | 穿越说明、世界观解释 |
| **情绪章** | 20-30% | 互动、搞笑、温情、心理戏 | 父子逛金陵、鱼王契约 |
| **推进章** | 30-40% | 事件发生、冲突升级、行动展开 | 百鲤跃龙门、炼丹修炼 |
| **高潮章** | 10-15% | 期待兑现、情绪宣泄、阶段性成果 | 化龙成功、打脸现场 |

### 3.3 情绪曲线设计

前10章的情绪走势（通用模式）：

```
高涨 ▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁▁→
      ↗           ↗        ↘
     /  \    ↗   /  \    ↗  \
    /    ↘  /   /    ↘  /    ↘
   /      ↘/   /      ↘/      ↘
  ▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔ 低落
   1    3    5    7    9    10章
  
  ↑开篇  ↑小爽点  ↑中压  ↑小高潮  ↑大高潮
 钩子   捡漏    训练   积累    兑现
```

- **第1章**：钩子向上（抓住读者）
- **第2-3章**：小爽点/喜剧/金手指验证（小幅上扬）
- **第4-6章**：积累阶段，中段压力（小幅下行或走平）
- **第7-8章**：准备工作完成，期待感蓄满（上扬）
- **第9-10章**：第一个大高潮兑现（最高点）

## 四、章节内结构

### 4.1 标准章节骨架

```
开篇（200-300字） → 承接前文/小过渡 → 引入本章核心事件
中段（1000-1500字）→ 事件发展/对话/推进 → 信息密度最高段
转折（300-500字）  → 小高潮/发现/冲突 → 本章情绪高点
结尾（200-300字）  → 钩子/悬念/下章预告 → 让读者想点下一章
```

### 4.2 黄金三章检查清单

- [ ] 第1章前500字内出现了钩子（悬念/笑点/冲突）
- [ ] 第1章内完成了"这书在讲什么"的基本认知
- [ ] 第2章内出现了金手指/核心设定/能力展示
- [ ] 第3章结尾有"想知道后续"的冲动
- [ ] 前三章完成了：主角身份 + 核心矛盾 + 金手指 + 第一章悬念

---

## 五、结构编排原则

1. **不要平均用力**：10章中需要有1-2章高潮章、2-3章情绪章，其余是推进和信息章
2. **情绪需要蓄力**：第1章和第9章不能都是高潮——中间需要下坡和平台期
3. **每个模式需要适配题材**：日常向用"日常切入型"开篇，历史文用"穿越＋世界观突变型"
4. **爽点和结构是配套的**：科研型推进天然适配解谜爽，倒计时型推进适配压迫破局爽
5. **前10章决定追读率**：必须在前10章内完成至少一次"蓄力→释放"的完整循环
